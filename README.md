# Overview
It's a simple React project to demonstrate interaction with backend. The project is not supposed to demonstrate knowlenge of React or any frontend technologies, rather it tries to show functionality of backend (PaymentGate-Backend). However, the project contains some business logic and manipulations with backend replies.
